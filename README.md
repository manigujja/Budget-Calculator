# Budget-Calculator

# Real-World Use Case: Weekly Budget Management for Students

A college student receives $100 weekly allowance and needs to track spending on:

1. Food and drinks 
2. School supplies 
3. Entertainment 
4. Transportation 

# Problem Statement

Without tracking, students often overspend early in the week and run out of money.

Create a program that helps students:

1. Know exactly how much money they have left
2. Check if they can afford items before buying
3. Calculate discounts to find better deals
4. View their spending patterns
